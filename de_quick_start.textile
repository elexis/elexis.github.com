title: "Einstieg: Elexis kennen lernen"
---

Sie müssen Java (Version 7 oder grösser) installiert haben. Downloads finden Sie unter "OpenJDK":http://openjdk.java.net/install/index.html oder von "Sun":http://www.java.com/de/download/. Unter Windows wird nur die 32-Bit-Version unterstützt, d.h. auch auf einem Windows 64-Bit müssen Sie die 32-Bit-Version von Java installiert haben, um Elexis ohne Probleme benützen zu können.

Zum ersten Kennenlernen empfehlen wir Ihnen die Installation der Demo-Datenbank. Dazu die Zip-Datei von "sourceforge":http://download.elexis.info/demoDB/demoDB_3.0_with_administrator.zip oder "elexis.info":http://sourceforge.net/projects/elexis/files/elexis%20full%20installation/3.0.0/demoDB_3.0_with_administrator.zip herunterladen.

In ihrem Heimatverzeichnis eine Ordner elexis anlegen. Die heruntergeladene Datei entpacken und den darin enthaltenen Ordner demoDB in dieses Verzeichnis elexis schieben.

Zuerst müssen Sie die ZIP-Datei für Ihr Betriebssystem herunterunterladen. Diese findet man unter
* "Sourceforge":https://sourceforge.net/projects/elexis/files/elexis%20full%20installation/3.0.0/
* "elexis.info":http://download.elexis.info/elexis.3.core/3.0.0/

Dann müssen Sie diese Datei entpacken. Meistens sollte dafür ein Doppelclick genügen.

Jetzt sollten Sie unter den entpackten Datein das Programm Elexis3 mit einem Doppelclick öffnen.

Benutzer von Windows können sich wahrscheinlich noch freuen, folgenden Hinweis zu quittieren.

!images/Install_10.png!

Falls Sie die DemoDB installiert haben, geben Sie gei der Aufforderung, einen Benutzernamen und ein Passwort einzugeben, beide Male "test" ein.

!images/Install_11.png!

Elexis öffnet, sich und zeigt einige Fenster an, nicht jedoch die Daten zu einem bestimmten Patienten.

!images/Install_12.png!

Um einen Patienten auszuwählen, klicken Sie auf den Reiter "Patienten".

!images/Install_13.png!

Um die Liste aller Patienten auszuwählen, klicken Sie einfach auf "Name".

!images/Install_14.png!

Nun können Sie auf eigene Faust Elexis erkundigen, oder sich das "Handbuch":http://sourceforge.net/projects/elexis/files/Handbooks/2.1.7.0/Handbuch/elexis.pdf/download (teilweise noch auf der Version 1.4 basierend) zu Gemüte führen.
